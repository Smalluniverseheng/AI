# AI 多模型对话平台

一个纯前端多模型 AI 对话平台，支持 15+ 家国内外厂商，单模型/多模型并行/辩论三种模式。

## 功能

- 🤖 15+ 家国内外 AI 厂商，100+ 模型
- 💬 单模型 / 📋 多模型并行 / ⚔️ 辩论（可配置轮数）
- 🧠 思考链展示（DeepSeek R1、o1、Claude 等）
- 📝 富文本渲染（Markdown、代码高亮、表格、LaTeX、Mermaid 图表）
- 🎤 语音输入（Web Speech API + MiMo ASR 兜底）
- 🔊 文字转语音播报
- 📎 文件/图片上传
- 💰 额度统计 + 预算告警
- 📱 PWA 可安装到桌面

## 使用

直接打开 `index.html` 即可，无需任何服务器。

## 部署到 GitHub Pages

1. Fork 或上传到 GitHub 仓库
2. Settings → Pages → Source 选 `main` 分支
3. 访问 `https://你的用户名.github.io/仓库名/`
